# Визуализация работы башенного крана

## Описание проекта
Веб-приложение для 3D визуализации и анализа работы башенного крана. Позволяет загружать данные о работе крана, просматривать их в интерактивном режиме и анализировать эффективность использования оборудования.

## Основные возможности

### 3D Визуализация
- Интерактивная 3D модель башенного крана
- Анимация движений в реальном времени
- Отображение путей движения крюка и грузов
- Визуализация зон работы (погрузка/разгрузка/бетон)
- Контроль безопасных параметров работы

### Анализ данных
- Определение типов грузов по весу
- Отслеживание времени операций
- Анализ эффективности использования
- Статистика по зонам работы
- Учет влияния ветровых нагрузок

## Технологии
- Three.js - 3D визуализация
- JavaScript (ES6+) - логика приложения
- HTML5/CSS3 - пользовательский интерфейс

## Установка

1. Клонировать репозиторий:
```bash
git clone https://github.com/Viremine/crane-visualization.git
cd crane-visualization
```

2. Установить live-server (если не установлен):
```bash
npm install -g live-server
```

3. Запустить сервер:
```bash
live-server
```

4. Открыть в браузере: `http://localhost:8080`

## Использование

### Визуализация данных
1. Загрузить файл с данными о работе крана
2. Настроить параметры в форме:
   - Размеры крана
   - Высота башни
   - Длина стрелы
   - Крепления к стене
3. Нажать "Применить" для запуска визуализации
4. Использовать элементы управления:
   - Воспроизведение/пауза
   - Скорость просмотра
   - Включение/отключение путей
   - Показ зон работы

### Формат входных данных
```json
{
  "timestamp": "2024-01-01T12:00:00Z",
  "hook": {
    "position": {"x": 0, "y": 10, "z": 0},
    "load": 1500
  },
  "wind": {
    "speed": 5,
    "direction": 180
  }
}
```
